# Medical-RAG
> 本專案旨在建置一個醫療問答 RAG（Retrieval-Augmented Generation）系統，透過結合醫療知識檢索與大型語言模型，提供準確且可追溯來源的醫療問答，以降低醫療人員在基礎諮詢上的負擔。
---
## 專案背景
> 由於醫療人員與病患之間的比例長期失衡，導致每位醫療專業人員需同時應對大量病患諮詢，使其在時間與精力上承受極大壓力。
 許多患者的問題屬於重複性高、屬於醫療知識查詢或基礎說明層級，若能透過智慧化系統進行初步回應，將可有效降低醫療人員的業務負荷，並提升醫療服務效率。
---
## 專案目標
1. **醫療專業知識導向回答能力**
   - 回答內容需基於醫療文件、指引或專業資料
   - 避免僅依賴語言模型的「記憶式生成」
2. **自然語言理解能力**
   - 能理解患者以口語、非專業術語提出的問題
   - 將患者問題正確映射至對應的醫療知識
3. **可追溯的回答來源**
   - 回答須基於檢索到的醫療文件
   - 提供來源或依據，降低幻覺風險
